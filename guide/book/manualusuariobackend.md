# Manual de Usuario Backend


## Seguridad JWT

Garantiza la seguridad de los endpoints.


### Autentification: 
Verifica las credenciales de un usuario que intenta acceder a los endpoints mediante un servicio de consumo de API tales como swagger-ui, postman, restfox, curl.
Usa las clases JRole, JCrendential, JUser definidas en JettraServer internamente utiliza la base de datos JettraServerDB que almacena los datos correspondientes.

### Acreditation:
	La acreditación la personaliza el usuario
